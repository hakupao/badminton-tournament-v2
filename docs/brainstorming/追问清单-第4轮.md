## V. 技术栈确认（我的推荐）

基于你的情况（有开发经验、想边做边学、6天时间、日本地区），我推荐：

```
前端：Next.js 14+ (React) + Tailwind CSS + shadcn/ui
后端：Next.js API Routes（或 Cloudflare Workers）
数据库：Cloudflare D1 (SQLite) 或 SQLite（取决于部署方案）
ORM：Drizzle ORM（轻量、类型安全）
部署：Cloudflare Pages / Vercel
语言：TypeScript
```

为什么选这些：
- **Next.js**: 前后端一体，不用分开两个项目，开发快
- **Tailwind + shadcn/ui**: 手机端适配方便，UI 组件开箱即用，界面好看
- **SQLite/D1**: 对这个规模最合适，简单可靠
- **TypeScript**: 类型安全，减少 bug
