# 功能需求讨论

## 待确认的核心问题

### 1. 比赛形式
- 单打 / 双打 / 混双？还是都要支持？
- 赛制：循环赛、淘汰赛、小组赛+淘汰赛？
- 每场比赛的计分规则（标准羽毛球 21 分制？还是简化版？）

### 2. 选手管理
- 大概多少人参加？（10人？20人？50人？）
- 是否需要注册/登录？还是一个人管理员录入就行？
- 选手信息需要哪些？（姓名、昵称、水平等级？）

### 3. 赛程安排
- 手动排赛程还是自动生成？
- 是否需要考虑场地数量（同时进行多场）？
- 是否需要时间安排？

### 4. 比分记录
- 谁来记录比分？（裁判？选手自己？管理员？）
- 实时记录还是赛后录入？
- 是否需要实时比分展示？

### 5. 排名与统计
- 积分榜 / 排行榜？
- 个人战绩统计？（胜率、得失分等）
- 历史赛事记录？

### 6. 其他可能的功能
- 赛事公告/通知
- 对阵表/赛程表可视化展示
- 分享功能（链接分享给朋友查看）
- 移动端适配（手机上用为主？）
